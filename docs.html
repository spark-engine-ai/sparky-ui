Installation

Get started with Spark UI in your project with these simple steps.

1. Install via NPM

npm install spark-ui

2. Add to Tailwind Config

// tailwind.config.js
module.exports = {
  plugins: [
    require('spark-ui')
  ]
}

3. Import CSS

/* In your CSS file */
@tailwind base;
@tailwind components;
@tailwind utilities;
