Get Started in Minutes

Three simple steps to start building ultra-modern interfaces with Spark UI.

1

Install Package

npm install -D spark-ui
2

Configure Tailwind

plugins: [require('spark-ui')]
3

Start Building

<button class="btn btn-primary">

Complete Example

<!-- Add to your tailwind.config.js -->
module.exports = {
  plugins: [require('spark-ui')]
}

<!-- Use in your HTML -->
<div class="card">
  <div class="card-body">
    <h2 class="card-title">Welcome to Spark UI</h2>
    <p>Beautiful components with glass effects</p>
    <button class="btn btn-primary">Get Started</button>
  </div>
</div>
